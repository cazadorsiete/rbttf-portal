# Proyecto de Protección de la RBTTF

Sitio web estático listo para Vercel.

## Cómo subirlo
1. Descomprime este ZIP.
2. En GitHub, dentro del repositorio `rbttf-portal`, usa **Add file > Upload files**.
3. Sube `index.html` y `README.md`.
4. En Vercel importa el repositorio.
5. Si pregunta framework, selecciona **Other** o deja automático.
6. Publica.

Cuando Vercel genere la URL, pégala en el iframe de ArcGIS Hub.
